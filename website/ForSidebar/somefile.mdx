## live theme codeblock

1. first run this code

> npm i @docusaurus/theme-live-codeblock 

2. then add a `themes` section in `docusaurus.config.js` file and add the name of the theme i.e.

> @docusaurus/theme-live-codeblock


```jsx live
function examplelive(props){
    const[ count , setCount ] = useState(0);

    return(
        <div>
        <h1>{count}</h1>
        <button onClick={() => setCount(count+1)}>ADD</button>
        </div>
    )
}
```
